Python
import numpy as np

def branchless_select(a, b, t):
	# equivalent to HLSL lerp(a, b, step(0.5, t))
	mask = (t >= 0.5).astype(float)
	return a + (b - a) * mask
C++
#include <cstdint>

inline float branchless_lerp(
	float a, float b, float t)
{
	// no branch, compiler-friendly
	return a + (b - a) * t;
}

09 // Render Pipeline Stages

The render frame decomposes into discrete sequential stages. Each stage has a fixed budget; exceeding any node stalls the GPU queue.

01
Cull & Gather
Frustum + occlusion culling. Surviving draw calls are batched into indirect argument buffers on the GPU timeline.
02
GBuffer Pass
Geometry stage writes BaseColor, WorldNormal, and Roughness into MRT targets.
03
Lighting & Shadows
Deferred lighting pass consumes the GBuffer. Ray-traced shadow maps are resolved and accumulated in a half-res buffer.
04
Post-Process & Resolve
TAA, bloom, and tone-mapping run as full-screen compute dispatches. Output resolves to the swap-chain backbuffer.

10 // Approach Comparison

Side-by-side analysis of two implementation strategies. Left column highlights the flawed or legacy approach; right column shows the preferred pattern.

❌ Divergent Branch
All 32 threads evaluate both branches then mask results.

— 2× ALU cost
— Pipeline stall on memory fetch
— Warp efficiency: ~50%
✓ Branchless Select
Use lerp(a, b, step(t, x)) to eliminate branch divergence entirely.

— Uniform control flow
— No warp serialization
— Warp efficiency: ~100%
